# Lekcije – Evidencija časova

Jedan fajl po lekciji. Nivoi: A1, A2, B1, B2.

## Struktura

```
lekcije/
├── A1/
│   ├── 1-1.md
│   ├── 1-2.md
│   └── ...
├── A2/
├── B1/
└── README.md
```

## Format lekcije

1. **Belške** – pišeš tokom časa (nemački ili mešovito)
2. **Prevodi** – sve rečenice ili tekstovi prevedeni [DE] [SR] [EN]
3. **Gramatika** – gramatičke tačke iz lekcije
4. **Reči** – linkovi na vocabulary/

Ispravke gramatike agent radi **svuda** – gde god pišeš (Belške, bilo koja sekcija).

## Workflow

1. Pišeš belške u fajl lekcije
2. Kažeš: "Ispravi belške iz lekcije X" ili "Ispravi i izvuci reči i gramatiku iz lekcije X"
3. Agent: ispravlja inline (gde god treba), dodaje prevode u Prevodi, izvlači reči u vocabulary/, beleži gramatiku u lekciju
