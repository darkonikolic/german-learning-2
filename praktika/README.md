# Praktika

Beleške iz praktičnih sesija (razgovor, vežbanje, konverzacija).

## Format

Jedan fajl po sesiji: `Y-m-d.md` (npr. `2025-03-11.md`).

```markdown
# Praktika – [datum]

## Šta sam radio
_(kratak opis – šta, sa kim, koliko)_

## Belške
_(fraze, greške, napomene, nova rečenica)_

## Prevodi
_(ako treba – [DE] [SR] [EN])_
```

## Workflow

1. Piši beleške tokom ili posle praktike
2. "Ispravi i izvuci iz praktike X" – agent ispravlja, dodaje prevode, izvlači reči u vocabulary/
