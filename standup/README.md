# Standup

Dnevni izveštaji. Jednostavni, kratki. Format: jedan pasus = [DE] [SR] [EN].

## Format imena

`Y-m-d.md` – dan kada se izveštaj priča (npr. `2025-03-12.md` za sutrašnji standup).

## Format sadržaja

```markdown
# Standup – [datum]

[DE]
1. {tačka 1}
2. {tačka 2}
3. {tačka 3}

[SR]
1. {tačka 1}
2. {tačka 2}
3. {tačka 3}

[EN]
1. {tačka 1}
2. {tačka 2}
3. {tačka 3}
```

**Uvek:** tačke sa brojevima (1, 2, 3…). Jedan blok [DE] [SR] [EN] sa istim brojem tačaka.
