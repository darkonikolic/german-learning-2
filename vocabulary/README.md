# Vocabulary – Extracted Words

Words are distributed by type in one file per type (no subfolders):

| File | Type | Examples |
|------|------|----------|
| Substantive.md | Nouns | die Arbeit, der Kollege, das Buch |
| Verben.md | Verbs (infinitive only) | gehen, machen, lesen |
| Adjektive.md | Adjectives | freundlich, wichtig |
| Adverbien.md | Adverbs | jeden Tag, sehr |
| Präpositionen.md | Prepositions | zur, am |
| Andere.md | Conjunctions, particles, pronouns | und, aber, ich |

Within each file words are sorted alphabetically. Format per word: `## {word}`, `[sr]`, `[en]`, empty line. Verb tenses are extracted separately – not in the vocabulary.
